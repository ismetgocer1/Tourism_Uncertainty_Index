This repository for explaining the Tourism Related Uncertainty Index creating process. 

You can find the system requirements in the Requirement.txt file.

Necessary Python codes are provided in the "Tourism_Uncertainty_Index_Notebook (Codes).ipynb" file.

Sample reports were given in the "Sample Reports" folder.

Attention!!!
You will need to use "en_core_web_lg" to connect relevant LLM.
To download the required spaCy language model, run the following command:

python -m spacy download en_core_web_lg

This will install the model locally so the code can run properly.




Index creation progress are explained in "Tourism_Uncertainty_Keywords_Number.xlsx" file and also Appendix of the "Introducing the New Tourism Uncertainty Index and the Sustainable Development Goals: A Context Window Approach" paper.
